Detailed Description

Generic Class, used to Filter LiveData objects.
Filters when LiveData is updated, works asynchronously.
Behavior is defined by it's FilterComponents.

de.datenkraken.datenkrake.ui.util.LiveDataFilter< T >.LiveDataFilter ( LiveData< List< T >>  filterAbleContent)

Constructor initialing important components components of this class and adding observer to given filterable content to omit it into liveDataInput. We have to use liveDataInput so we can notify it's observer via MutableLiveData#postValue(Object). Also ads an androidx.lifecycle.Observer to liveDataInput to filter as soon as the value changes. It filters by iterating over filterComponents and calling FilterComponent#checkForValidity(Object) for each T in the List of Ts in the given LiveData.
