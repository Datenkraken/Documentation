Detailed Description

This ViewModel is responsible for fetching GetCategoriesOnlyQuery.Categorys and GetCategoriesQuery.RssSources from the server and providing them to the views.
It also retrieves Sources from the androidx.room.Database that have already been added by the user to find out which ones should be displayed in the recommendation page.
It is also used to temporarily store added GetCategoriesOnlyQuery.Categorys and GetCategoriesQuery.RssSources so the latter can be added after submission by the user.
It is shared among the classes in the recommendation package.
