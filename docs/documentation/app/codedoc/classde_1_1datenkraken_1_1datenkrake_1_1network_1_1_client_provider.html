Client<?> de.datenkraken.datenkrake.network.ClientProvider.instantiateClient ( Class<? extends Client<? extends ITask >>  clientClass) throws NoSuchMethodException, IllegalAccessException, InstantiationException, InvocationTargetException

Tries to instantiate the given class of a Client.
Searches through all accessible constructors of the given class prioritizing constructors accepting a ClientProvider.
Tries to use the default constructor if no prioritized constructor was found.
