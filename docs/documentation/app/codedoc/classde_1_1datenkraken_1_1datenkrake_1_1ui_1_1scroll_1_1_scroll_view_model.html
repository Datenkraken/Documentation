Detailed Description

Class provides the Articles to be shown in the ScrollFragment. It is shared with the de.datenkraken.datenkrake.ui.singlearticle.ArticleViewFragment.
Provides the functionalities to retrieve a next article, get filtered articles and save articles. Can also set articles to read.
