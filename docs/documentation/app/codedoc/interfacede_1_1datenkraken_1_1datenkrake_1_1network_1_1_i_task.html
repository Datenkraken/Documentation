Detailed Description

Interface providing functions to represent a task for HTTP Queries. To start or cancel this task call request() or cancel(). This class should be implemented by template Tasks for different Clients. When request() is called, this class will be distributed by TaskDistributor and executed by it's the client, given in processedBy().
