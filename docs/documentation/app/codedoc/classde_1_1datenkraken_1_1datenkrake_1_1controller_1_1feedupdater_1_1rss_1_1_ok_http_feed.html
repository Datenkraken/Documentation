Detailed Description

Class containing necessary functionality to request and parse a RSSFeed.
This class should be used by instantiating an anonymous subclass and defining the required functions. To start or cancel this task call request() or cancel(). This class can be extended to perform more specialized tasks. When request() is called, this instance will be executed by the de.datenkraken.datenkrake.network.TaskDistributor. Provides functionalities to parse SyndFeeds and build and handle Requests.

Parses the given content to HTML. Removes all image tracker and tries to extract a front page image from the rest of images.
The first image found is used as the front page image.
The found front page image is removed from the given content of the Article.
If it doesn't find any image tags in the parsed content, it will return the original content so non HTML content will stay the same.
Uses HTMLUtil#removeImageTracker(Elements, int) to remove trackers from the content and HTMLUtil#getTitleImage(Elements) to get the image.

List<Article> de.datenkraken.datenkrake.controller.feedupdater.rss.OkHttpFeed.parseArticles ( SyndFeed  feed)
private

Parses given SyndFeed to an Article list.
Extracts informations, such as title, author, published date, description, link, content and image from each entry of the feed and creates an article object.
The image gets extracted form the content or the description, using extractImageAndRemoveTracker(String, Article) and the resulting string is set as content or description.
These objects gets collected and returned as a list.
Standardizes the articles using standardizeArticle(Article).

Parameters
feedSyndFeed to be used to get the Articles.
Returns
List of Articles from the feed.
