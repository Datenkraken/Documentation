Detailed Description

Class providing functionality to save and load serializable objects.
Used to save the output from IEventProcessor and de.datenkraken.datenkrake.surveillance.background.IBackgroundProcessor via the PacketSaver to the disc. Will be loaded again by de.datenkraken.datenkrake.surveillance.background.PacketListStream.
