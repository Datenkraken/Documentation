Detailed Description

Generic event class that is used for passing events via live data. The Event carries a content, that can be accessed via getContentIfNotHandled or peekContent.

◆ getContentIfNotHandled()

T de.datenkraken.datenkrake.util.Event< T >.getContentIfNotHandled ( )

Get content of the event if the event has not been handled yet (handled is false). After accessing the content, handled will be set to true. Thus, the content can not be accessed again with this function If the content already has been handled, this function returns null.

Returns
the content of the event or null if the event has been handled.
