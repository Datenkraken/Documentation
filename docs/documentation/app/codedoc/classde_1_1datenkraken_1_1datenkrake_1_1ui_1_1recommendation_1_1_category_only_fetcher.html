Detailed Description

Class used to fetching a list of GetCategoriesOnlyQuery.Category first to retrieve the names.
It uses an instance of CategoryFetcher to fetch the entire GetCategoriesQuery.Categorys and GetCategoriesQuery.RssSources list after the category fetching has been successful.
It then provides this list to the RecommViewModel. This is done to achieve faster access speed.
