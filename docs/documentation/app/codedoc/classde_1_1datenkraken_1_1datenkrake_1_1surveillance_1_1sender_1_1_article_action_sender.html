Creates and returns the ITask, which will be send to the server.
When this method gets implemented in a subclass, it should not access any member fields in its class. The returned ITask should send all given packets. Callback#onSuccess() must be called in the successful callback of the created ITask or in this function to signalize, that the saved packets can be deleted.
